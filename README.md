# Darukaa Earth: Carbon & Biodiversity Dashboard

## Overview
I developed a geospatial data visualization dashboard focused on environmental project monitoring.

The application allows users to interact with a map interface, add project sites dynamically, and visualize environmental metrics such as carbon levels through integrated charts.

## Features
- Interactive map using Leaflet
- Click to add project sites (markers)
- Display site coordinates
- Data visualization using Chart.js


## Tech Stack
- React
- Leaflet (for mapping)
- Chart.js (for data visualization)

## Approach
Due to time constraints, I focused on implementing core geospatial interaction and data visualization features. The system is designed to scale with backend integration, authentication, and advanced geospatial analytics using PostGIS.

## Future Improvements
- Backend integration (Flask/Django)
